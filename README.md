# Matrix Calculator (C++23)

Team C++23 project: console matrix calculator with modular architecture.

## Planned structure

- src/core — matrix model and linear algebra logic
- src/io — input parsing and formatted output
- src/util — shared utility types/helpers
- src/report — report steps and orchestration
- src/app — application entry point
- tests — unit tests

## Planned project goals

- C++23
- OOP architecture
- abstract base class and polymorphism
- CMake build system
- unit tests
- modern C++ features:
- std::unique_ptr
- std::shared_ptr
- std::optional
- std::variant / std::visit
- RAII
- constexpr
- exceptions
